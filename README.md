# pytorch-raytracing
Ray tracing written in Pytorch, which uses Nvidia GPUs through CUDA.

The Jupyter Notebook, and its previous versions, are available at https://www.kaggle.com/code/novakstijepic/fork-of-pytorch-ray-tracing .
